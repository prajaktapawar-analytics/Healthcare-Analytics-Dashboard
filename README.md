# 🏥 Healthcare Analytics Dashboard

An end-to-end **Healthcare Analytics** project built using **Python, SQL, Excel, and Power BI** to analyze patient data, hospital performance, treatment costs, and key healthcare KPIs. This project demonstrates the complete data analytics workflow from data preparation to interactive dashboard creation.

---

## 📌 Project Overview

This project analyzes healthcare data to provide valuable insights into patient demographics, admission trends, disease distribution, treatment costs, and hospital performance through interactive dashboards and reports.

---

## 🚀 Technologies Used

- Python
- SQL (SQLite)
- Microsoft Excel
- Power BI
- Git & GitHub

---

## 📊 Key Features

- Data Cleaning & Preprocessing
- Patient Demographics Analysis
- Admission Trend Analysis
- Treatment Cost Analysis
- Healthcare KPI Reporting
- SQL Data Analysis
- Interactive Power BI Dashboard
- Business Insights

---

## 📈 Key KPIs

- Total Patients
- Total Admissions
- Average Treatment Cost
- Average Length of Stay
- Disease Distribution
- Department-wise Admissions
- Monthly Admission Trends
- Patient Recovery Rate
